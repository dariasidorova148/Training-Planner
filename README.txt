# Training Planner (План тренировок)


Приложение для планирования и отслеживания тренировок с фильтрацией и сохранением данных.


## Автор

[Даша Сидорова]

## Описание программы


Training Planner — это десктопное приложение для учёта спортивных тренировок. Оно позволяет:

* Добавлять тренировки с указанием даты, типа и длительности.
* Фильтровать записи по типу тренировки и дате.
* Сохранять все данные в JSON‑файл для постоянного хранения.
* Просматривать историю тренировок в удобной таблице.

## Требования для запуска

* Python 3.6 или выше.
* Библиотека `tkinter` (входит в стандартную поставку Python).
